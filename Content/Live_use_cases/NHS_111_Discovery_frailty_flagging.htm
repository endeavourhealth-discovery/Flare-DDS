NHS 111 Discovery frailty flagging

Note: For the latest frailty statistics see the DDS wiki page - NHS 111 Discovery frailty flagging

Project summary

The NHS 111 Discovery frailty flagging project utilises Discovery data to flag key information to NHS 111 call handlers to improve the patient pathway and experience, and reduce the unnecessary use of emergency services and admission to A&E.

Project population

All patients within Inner East London; City and Hackney, Tower Hamlets, Waltham Forest, and Newham CCGs.

Project cohort

Frail patients.

Project criteria

See the Frailty algorithm wiki page for more details.

Data state

Patient identifiable.
